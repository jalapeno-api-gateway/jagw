About Properties#

When making a request or a subscription you can specify properties (i.e. in TopologyRequest or TelemetrySubscription). This allows you to define the properties you want to be returned for the requested objects. This is optional, since omitting properties results in all available properties being returned.

Different objects provide different properties (i.e. LSNode, LSLink, TelemetryData).

invalid properties

The Jalapeño API Gateway ignores invalid properties. Make sure therefore, that the strings you provide match the properties below exactly.
